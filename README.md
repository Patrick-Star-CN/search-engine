# search-engine
字节跳动青训营大作业

# 结构目录说明

```
├─app               // 应用层面的业务代码
│  ├─apiExpection   // api返回的错误
│  ├─controllers    // 控制器
│  ├─midwares       // 路由中间件
│  ├─models         // gorm的模型
│  ├─services       // DAO
│  └─utils          // 常用功能性代码
│
├─config            // 配置层面的业务代码
│  ├─config         // 配置文件绑定
│  ├─database       // 数据库绑定
│  └─router         // 路由设置
│
└─doc               // 文档
```
